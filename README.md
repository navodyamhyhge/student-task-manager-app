# Student Task Manager

A simple web application to manage student tasks. Users can add, delete, and mark tasks as completed. The UI design is based on a Figma layout.

---

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive and user-friendly design

---

## Technologies Used
- HTML
- CSS
- JavaScript
